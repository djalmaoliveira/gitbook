O Livro da Comunidade Git

Bem vindo ao Livro da Comunidade Git. Esse livro tem sido construido por dúzias de pessoas dentro da comunidade Git, e é principalmente para ajudar a você a aprender como usar o Git tão rapidamente e facilmente quanto possível.

Se você achou alguma coisa desatualizada, dê uma sugestão sobre como melhorá-lo, ou se gostaria de ajudar com o livro, por favor veja a página Como contribuir, ou envie uma nota para nosso mantenedor.
